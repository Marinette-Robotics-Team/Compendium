Robot CodeÂ¶

Most of what software does is write robot code that does some desired task. Thus, understanding how to write robot code is the most important technical skill to learn as a software member.

This guide assumes that the reader has knowledge of java basics, such as objects and classes, public and private items, data types, conditionals, and the like.
