Integrating code
================

This article describes how to integrate different pieces of code written by different people, usually in different projects, into one cohesive project. Although this process is also called merging, if you're looking for how to use SVN merging to solve conflicts, see `Resolving SVN Conflicts <svn/svnconflict.html>`_.
